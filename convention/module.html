모듈 가이드문서 자주 사용하는 모듈 예시와 가이드

자주 사용하는 모듈의 기본적인 html, css, js구성에 대해 설명하고 예제를 제공합니다.
하기 구성을 참고해 프로젝트에 적용할 수 있습니다.

예제 섹션의 샘플들은 모두 필수적인 디자인만 되어 있습니다.

체크박스 디자인

하기 템플릿 부분은 common.css에 포함되어 있습니다.

HTML

input[type=checkbox]와 label을 마크업한 뒤, 실제 체크박스는 숨깁니다.
체크박스를 label:before의 background속성으로 대체하고, 체크박스가 checked된 경우도 해당 이미지로 변경합니다.

CSS

common.css에 가장 기본적인 css가 선언되어 있습니다.
디자인에 따라 background속성을 URL로 변경하거나 width, height, position 값 등을 재조정합니다.

						.checkbox + .label {cursor:pointer;}

						/* 진짜 체크박스 숨기기 */
						.checkbox {overflow:hidden; position:absolute; width:1px; height:1px; padding:0; border:0; margin:-1px; clip:rect(0 0 0 0);}

						/* 가짜 체크박스 디자인 */
						.checkbox + .label:before {display:inline-block; width:10px; height:10px; border:1px solid #dfdfdf; margin-right:5px; content:'';}

						/* 체크박스에 체크되었을 때 스타일 */
						.checkbox:checked + .label:before {background:#bfbfbf;}

라디오버튼 디자인

하기 템플릿 부분은 common.css에 포함되어 있습니다.

HTML

input[type=radio]와 label을 마크업한 뒤, 실제 라디오버튼은 숨깁니다.
라디오버튼을 label:before의 background속성으로 대체하고, 라디오버튼이 checked된 경우도 해당 이미지로 변경합니다.

CSS

common.css에 가장 기본적인 css가 선언되어 있습니다.
디자인에 따라 background속성을 URL로 변경하거나 width, height, position 값 등을 재조정합니다.

					.radio + .label {cursor:pointer;}

					/* 진짜 라디오버튼 숨기기 */
					.radio {overflow:hidden; position:absolute; width:1px; height:1px; padding:0; border:0; margin:-1px; clip:rect(0 0 0 0);}

					/* 가짜 라디오버튼 디자인 */
					.radio + .label:before {content:''; display:inline-block; margin:0 5px 0 0; width:13px; height:13px; border:1px solid #dfdfdf; border-radius:13px; text-align:center; vertical-align:middle; }

					/* 라디오버튼에 체크되었을 때 스타일 */
					.radio:checked + .label:before {background:#bfbfbf;}

파일첨부 디자인

하기 템플릿 부분은 common.css에 포함되어 있습니다.

HTML

부모 div 안에 input[type=text]와 input[type=file], label을 마크업한 뒤, 실제 파일찾기 버튼은 숨깁니다.
label을 파일첨부 버튼으로 사용합니다.

CSS

common.css에 가장 기본적인 css가 선언되어 있습니다.
디자인에 따라 width, height, position 값 등을 재조정합니다.

						.file_upload {position:relative; width:260px;}
						.file_upload .file_route {width:200px; height:28px; border:1px solid #bfbfbf;}

						/* 진짜 파일첨부 버튼 숨기기 */
						.file_upload .file {position:absolute; width:1px; height:1px; opacity:0; overflow:hidden;}

						/* 가짜 파일첨부 버튼 디자인과 위치 조정 */
						.file_upload .label {display:inline-block; position:absolute; top:0; right:0; height:28px; padding:7px; background-color:#bfbfbf; color:#fff; cursor:pointer;}

common.js에 선언되어 있습니다.

						// lnb 탭이동 2depth, 활성화된 class에 view 추가
						$('.lnb > ul > li > a').on('click', function (){
							if (!$(this).parent().find('ul').is(':visible')){
								$('.lnb > ul > li.view').removeClass('view');
								$(this).parent().addClass('view');
							} else {
								$(this).parent().removeClass('view');
							}
						});

						// lnb 탭이동 3depth 활성화된 class에 on 추가
						$('.lnb > ul > li > ul > li > a').on('click', function (){
							var index = $(this).parent().index();
							if (!$(this).parent().find('ul').is(':visible')){
								$('.lnb > ul > li > ul > li.on').removeClass('on');
								$(this).parent().addClass('on');
							} else {
								$(this).parent().removeClass('on');
							}
						});
